# Лабораторная работа по C++ (тестовая)

Это тестовая лабораторная работа, разработанная в рамках учебного задания. Проект написан на языке C++ и демонстрирует базовые принципы программирования, структуры данных и работу с системой контроля версий Git.

## Цель работы
- Освоение базовых конструкций языка C++
- Практика работы с Git и GitHub
  
## Требования
- Компилятор C++ (g++ 11+, Clang 14+ или MSVC 2022+)
- Git для контроля версий
