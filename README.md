# Проект "Пример README"

Добро пожаловать в демонстрационный проект!  
Здесь показаны различные варианты форматирования текста в Markdown.

---

## Форматирование текста

- *Курсивный текст* можно использовать для выделения слов.
- **Полужирный текст** помогает акцентировать внимание.
- ***Комбинированный стиль*** — одновременно курсив и жирный.
- ~~текст~~ используется для выделения текста.

---

## Списки

### Ненумерованный список
- Первый элемент
- Второй элемент
- Третий элемент

### Нумерованный список
1. Шаг первый
2. Шаг второй
3. Шаг третий
